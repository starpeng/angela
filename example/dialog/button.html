dialog

对象框中带按钮，默认点击按钮之后会触发关闭对话框，若不想关闭对话框，return false。调用代码如下所示：
$("#demo").dialog({
position : "center",
modal : true,
draggable : true,
buttonSet : {
"确定" : function()
{
alert('确定');
return false;
},
"取消" : function()
{
alert('取消');
}
}
});
