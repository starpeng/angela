jQuery网格插件 jqGrid

主要特点:
-Full control with JavaScript API
-Data returned from the server is XML
-Simple configuration
-Ability to load big datasets (paging)
-Resizable columns
-Server-side sorting
-Support of links, images, checkboxes
-You can add more than one grid on a sigle page (master-detail)

jsTree是一个 基于jQuery的Tree控件。支持XML，JSON，Html三种数据源。提供创建，重命名，移动，删除，拖"放节点操作。可以自己自定义创建，删 除，嵌套，重命名，选择节点的规则。在这些操作上可以添加多种监听事件。
