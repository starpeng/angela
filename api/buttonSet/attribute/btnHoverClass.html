buttonSet.btnHoverClassattribute

Version Added:0.6.0

Type:String

Default Value:"ui40LightBlueGradientBG"

Description:用于设置鼠标滑过button时类名

HTML

<div id="demo" style="margin:10px;">
    <span class="uiRadio" group="abc">测试1测测试1测11</span><span class="uiRadio" group="abc">测试2</span><span class="uiRadio" group="abc">测试3</span><span class="uiRadio" group="abc">测试4</span>
</div>

CODE

$("#demo").buttonSet({
    btnHoverClass : "ui40CadetBlueGradientBG"
});
测试1测测试1测11测试2测试3测试4
