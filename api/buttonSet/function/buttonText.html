functionbuttonSet.buttonText

Version Added:0.6.0

return :jQuery, String

arguments:buttonSet.buttonText(index,[text])

参数说明：

index(Number):指定button位置

text(String):用于修改button的字符串

Description:根据index获取或设置buttonSet里button对象的文本

HTML

<div id="demo" style="margin:10px;">
    <span class="uiRadio" group="abc">测试1测测试1测11</span><span class="uiRadio" group="abc">测试2</span><span class="uiRadio" group="abc">测试3</span><span class="uiRadio" group="abc">测试4</span>
</div>

CODE

$("#demo").buttonSet();
console.debug($("#demo").buttonSet("buttonText", 1));
$("#demo").buttonSet("buttonText", 1, "文本修改")
