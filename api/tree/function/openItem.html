functiontree.openItem

Version Added:0.6.0

return :jQuery

arguments:tree.openItem(dest)

参数说明：

dest(Array, jQuery):用于指定打开的位置或元素

Description:打开dest指向的元素

HTML

<div id="demo" style="margin:20px;"></div>

CODE

$("#demo").tree({
      url : "../../tree.json",
      select : function(item, e)
      {
          var arr = $("#demo").tree("getItemIndexArray", item);
          $("#demo").tree("openItem", arr);
      },
      getMoreData : function(indexArray ,callBack)
      {
          $.getJSON("../../tree.json", function(data)
          {
               callBack(data);
          });
      }
});
