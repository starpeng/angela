functionprogressBar.scroll

Version Added:0.6.0

return :jQuery,Number

arguments:progressBar.scroll(scrolling)

参数说明：

scrolling(Boolean):指定progressBar是否滚动

Description:设置滚动条滚动状态（type必须为"scroll"），scrolling为true滚动，为false停止滚动

HTML

<div id="demo" style="width:400px;height:10px;">
</div>

CODE

$("#demo").progressBar({
    type : "scroll",
    scrollTime : 500
});
$("#demo").progressBar("scroll", false);
window.globalTimer = setTimeout(function()
{
    $("#demo").progressBar("scroll", true);
}, 1000);
