slide.clickattribute

Version Added:0.6.0

Type:Function

Default Value:null

Description:设置点击slide时调用的方法，return false可阻止默认实现。

HTML

<div id="demo" style="width:400px;height:10px;margin:10px;"></div>

CODE

$("#demo").slide({
    click : function()
    {
        console.debug("click");
        return false;
    }
});
