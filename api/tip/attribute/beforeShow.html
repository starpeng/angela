attributetip.beforeShow

Version Added:0.6.0

Type:Function

Default Value:null

Description:tip显示前调用该函数，若函数return false可阻止默认实现（显示tip），this指向tip对象，target指向tip提示的目标对象，e是jQuery封装的event事件参数

HTML

<div id="demo" style="width:400px;margin:150px;border:1px solid red;">
                <p>
                    我的tip测试
                </p>
            </div>
            <div id="tipDemo" style="width:200px;">消息提示消息提示消息提示</div>

CODE

$("#tipDemo").tip({
      target : $("#demo"),
      beforeShow : function(target, e)
      {
          return false;
      }
});
